# Luyen code theo mau

Ung dung Streamlit giup luyen go code theo file mau `.py` hoac `.ipynb`.

## Chay ung dung

```powershell
streamlit run app.py
```

## Tinh nang

- Tai file mau `.py` hoac `.ipynb`.
- File `.ipynb` duoc tach theo tung code cell de luyen rieng.
- Hien mau code ben trai va vung go lai ben phai.
- So sanh trong luc go bang khung loi theo dong.
- Mau do: ky tu sai hoac thieu.
- Mau vang: ky tu thua.
- Dong ho bam gio voi nut bat dau, dung, dat lai.
- Tuy chon bo qua khoang trang cuoi dong hoac dong trong.
